# Flip7
Flip7 for teaching students how to write a player's code

I was teaching my 6th graders how to program in Python and asked them to write player code for the Flip7 card game

This is an all-in-one html page that provides a front-end, allows students to enter their python code, and allows humans to play against their code or do simulations to see which player's code wins the most. All done with Claude

Just download the html file and open it up in a browser. The functions that the students have to implement are described in the html page.
